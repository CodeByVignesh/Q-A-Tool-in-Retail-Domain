### Q&A Tool in Retail Domain

Technologies used are

Lang Chain is the framework for building LLM Applications
Hugging Face is an LLM Library
Streamlit is an Python Framework for building UI Applications
Chroma is an Vector DB for storing Vectors
MySQL is an relational DB for storing structured data

Retail Store that has stored the data in the MySQL DB. 

Flow

UI -> Question -> LLM -> Convert Question to SQL Query format -> Fetch Data from SQL -> Give the answer to user in Text format
